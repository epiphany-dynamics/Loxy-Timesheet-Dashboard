# Automated Timesheet Dashboard

A timesheet management system built for a home care company. Employees log hours through a clean web interface, and administrators review and manage submissions from a centralized dashboard.

## What It Does

- **Employee Timesheet Entry** — Simple form-based time logging with validation
- **Admin Dashboard** — Review, approve, and manage all employee timesheets
- **Authentication** — Secure login with role-based access (employee vs. admin)
- **Real-Time Data** — Live updates powered by Supabase
- **Server Actions** — Next.js 14 server actions for secure data mutations

## Tech Stack

- **Next.js 14** — App Router with server actions
- **TypeScript** — Type-safe throughout
- **Tailwind CSS** — Responsive UI
- **Supabase** — Auth, database, and real-time subscriptions
